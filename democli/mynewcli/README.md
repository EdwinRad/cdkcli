# Usage
<!-- usage -->
```sh-session
$ npm install -g mynewcli
$ mynewcli COMMAND
running command...
$ mynewcli (-v|--version|version)
mynewcli/0.0.0 darwin-x64 node-v13.10.1
$ mynewcli --help [COMMAND]
USAGE
  $ mynewcli COMMAND
...
```
